# Context Tetris

Tetris meets LLM context windows — pack tokens into the context window.

Live at [mrunreal.github.io/ContextTetris](https://mrunreal.github.io/ContextTetris)

## Features
- Full Tetris with SRS rotation & wall kicks
- 7-bag randomizer, ghost piece, hold
- 10 LLM-themed levels (GPT-3.5 → AGI Mode)
- Mobile touch controls
- "CONTEXT OVERFLOW" game over
